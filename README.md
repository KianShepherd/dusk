## Description

A simple programming language built from scratch.

Uses flex as a tokenizer.

Uses YACC as a parser generator.

## Installation

Ensure you have flex and libfl-dev installed

`sudo apt install flex libfl-dev`

Then run the build script

`./build.sh`

To run the binary

`./dusk`
